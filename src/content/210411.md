- fetching

  - axios를 이용해 fetching을 간단하게 할 수있다.

    ```terminal
    >> npm install axios
    ```

    - 사용

    ```js
    import axios from "axios";
    //axios를 임포트

    class ComponentName extends React.Component {
      state = {};

      //axios를 사용해 fetch하는 함수
      getMovies = async () => {
        const 변수 = await axios.get(JSON);
        this.setState(state에 반영);
      };
      //async 비동기. 함수안의 await를 완료할때까지 기다린다.

      componentDidMount() {
        this.getMovies();
      }
      //JSON을 마운트

      render (){
        return ;
      }
    }
    ```
