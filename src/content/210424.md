##### Django

-   나는 왜 장고를 공부하는가? 왜 장고를 선택했는가?

    -   빠르게 웹을 개발하기위해
    -   파이썬과연결해 웹에서 파이썬의 코드를 사용하기위해
        -   플라스크를 사용해되 되지만 기본적인 기능등은 장고를 이용해 빠르게 해결하고 싶다.

-   장고

    -   The web framework for perfectionists with deadlines.

-   Divide and Conquer!

-   Django Config

    -   기본적인 장고앱
    -   장고의 설정과 관련된 사항을 이쪽에서 컨트롤한다.
        -   settings.py : 앱의 설치, 서드파티앱의 설치, 스태틱파일의 관리, 언어, 타임존등 각종 세팅
        -   urls.py : config App의 urls는 각 앱의 urls를 임포트해 전체적 관리를 하고 각 url은 각 앱에서 관리

-   Django App
    -   urls.py
    -   views.py : 말 그대로 view. template과 연결, template에서 사용하는 쿼리데이터를 관리.
    -   models.py : 데이터베이스 관리 테이블의 형태를 정의.
    -   admin.py : 장고어드민에서 보여주는 구조를 정의.

*   Django Shell
    -   기본적인 쿼리데이터들의 CRUD가 가능
